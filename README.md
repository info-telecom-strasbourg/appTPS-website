<h1 align="center"> The official Laravel website for the InsidePSBS app of <br> Télécom Physique Strasbourg </h1>

<br>

## To run this app : 

1. Clone the repo from GitHub
2. Run `composer install`
3. Copy the .env
4. In dev : run `php artisan serve`
