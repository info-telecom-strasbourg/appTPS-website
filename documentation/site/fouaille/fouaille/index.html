fouaille

Renvoie les informations fouaille de l'utilisateur connecté (grace au token).``

Requête

protocole methode url token
https GET /api/fouaille oui

Paramètres

Passage de parametre dans l'url pour la pagination exemple : https://app-pprd.its-tps.fr/api/fouaille?per_page=10&page=2
